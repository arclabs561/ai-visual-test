## You Haven't Been Invoked Yet - Here's How to Start!

The human annotation system is **ready for you to use**. Just run:

```bash
npm run annotate
```

**That's it!** Follow the prompts and start annotating.

---

## What This Does

1. **Finds samples** that need annotation
2. **Creates tasks** for you automatically
3. **Guides you** through each annotation
4. **Shows VLLM judgments** for comparison (if available)
5. **Saves your annotations** for validation

---

## Quick Commands

```bash
# Start annotating (simplest)
npm run annotate

# Full workflow menu
npm run annotate:full

# Validate your annotations
npm run validate:annotations

# Match with VLLM for calibration
npm run match:vllm
```

---

## What You'll Be Asked

1. **Your name/ID** (e.g., "alice")
2. **How many tasks** to annotate
3. **For each task**:
   - Score (0-10)
   - Issues (list problems)
   - Reasoning (explain your score)
